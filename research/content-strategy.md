# 에이전트 클래스 사이트 콘텐츠 전략서 v4

> **대상**: Agent Class Site 랜딩페이지 리디자인 프로젝트
> **작성 목적**: Apple.com 수준의 콘텐츠 밀도와 메시지 계층 구조를 위한 전략적 가이드
> **분석 기반**: 8개 챕터 메타데이터, 6개 쇼케이스, 기존 homepage.ts, 실제 마크다운 콘텐츠

## 1. 타겟 페르소나 프로필 (4종)

### 1.1 주니어 개발자 (Junior Dev)

**프로필**
- 연령: 25-32세, 경력: 1-3년차 백엔드/프론트엔드 개발자
- 기술 스택: Python 기초, API 연동 경험 보유
- 현재 상황: 단순 CRUD 개발에서 벗어나고 싶지만 AI 분야 진입 장벽이 높음

**페인 포인트**
1. "AI 에이전트를 만들어보고 싶지만 어디서부터 시작해야 할지 모르겠어"
2. "LangChain, RAG, MCP 같은 용어는 들어봤는데 실무에서 어떻게 쓰는지 감이 안 잡혀"
3. "ChatGPT API 호출 정도만 할 줄 아는데, 이걸로는 차별화가 안 돼"
4. "포트폴리오에 넣을 만한 AI 프로젝트가 필요해"

**추천 챕터**: Ch1 (오리엔테이션), Ch4 (n8n 기초), Ch5 (n8n RAG), Ch7 (MCP 서버)

**메시지 톤**
- "API 호출만 할 줄 아는 개발자 → 자율 에이전트를 설계하는 AI 엔지니어"

### 1.2 기획자/PM (Product Manager)

**프로필**
- 연령: 28-40세, 직무: PM/서비스 기획자
- 배경: 개발 경험 없거나 최소한의 코딩 지식
- 현재 상황: AI 자동화 프로젝트를 기획/관리해야 하는데 기술적 이해도가 부족

**페인 포인트**
1. "개발팀과 소통할 때 AI 에이전트가 뭔지 물어보는 게 창피해"
2. "AI 자동화로 업무 효율을 올리고 싶은데 어떤 업무가 자동화 가능한지 판단 불가"
3. "워크플로우를 기획할 때 기술적 제약사항을 몰라서 비현실적인 요구"

**추천 챕터**: Ch1, Ch2 (텍스트 LLM), Ch3 (이미지 LLM), Ch6 (CRM & MCP)

**메시지 톤**
- "반복 업무에 시간을 쏟는 팀 → 워크플로우 자동화로 생산성 3배 향상"

### 1.3 학생/취준생 (Student/Job Seeker)

**프로필**
- 연령: 20-26세, 컴공 재학생/부트캠프 수료생/취준생
- 현재 상황: 취업 시장에서 차별화할 포트폴리오가 없음

**페인 포인트**
1. "이력서에 'To-Do 앱 만들기' 프로젝트밖에 없어"
2. "AI 분야로 가고 싶은데 학교에서는 전통적인 알고리즘만 가르쳐줘"
3. "취업 공고에 LangChain, RAG, 벡터DB 키워드가 자꾸 나오는데 뭔지 모르겠어"

**추천 챕터**: Ch1, Ch2, Ch4 (n8n 기초), Ch8 (GIS)

**메시지 톤**
- "포트폴리오가 부족한 취준생 → 실전 프로젝트 6개를 보유한 AI 전문가"

### 1.4 창업자/스타트업 (Entrepreneur)

**프로필**
- 연령: 30-45세, 스타트업 창업자/1인 개발자/프리랜서
- 현재 상황: AI 기반 서비스를 빠르게 MVP로 만들고 싶음

**추천 챕터**: Ch4, Ch5 (n8n RAG), Ch6 (CRM & MCP), Ch7 (MCP 서버)

**메시지 톤**
- "AI 서비스를 빠르게 MVP로 만들고 싶은 창업자를 위한 실전 가이드"

## 2. 메시지 계층 구조 (Message Hierarchy)

### Level 1: Hero Headline (최상위 — 3초 임팩트)

**최종 추천 (Option D — 하이브리드)**
```
AI 에이전트로 시작하는 커리어 전환
8개 챕터 · 6개 프로젝트 · 33시간
```

**대안 옵션**
- A (포괄적): "AI 에이전트 마스터 클래스 / 8개 챕터와 6개 프로젝트로 완성하는 AI 에이전트 개발 여정"
- B (결과 중심): "스스로 생각하고, 도구를 사용하며, 문제를 해결하는 에이전트를 만드세요"
- C (변화 중심): "API 호출만 하는 개발자에서 자율 에이전트를 설계하는 AI 엔지니어로"

### Level 2: Section Headings (감성 + 정보)

| 섹션 | 제목 | 감정 유발 |
|-----|------|---------|
| Why Section | "왜 지금 AI 에이전트인가?" | 호기심 + 시급성 |
| Value Props | "8주 후, 당신이 만들 수 있는 것들" | 열망 + 구체성 |
| Curriculum | "기초부터 실전까지, 단계별 학습 여정" | 안정감 + 체계성 |
| Personas | "당신의 상황에 맞는 학습 경로" | 공감 + 맞춤형 |
| Showcases | "실제 작동하는 AI 에이전트를 체험하세요" | 신뢰 + 체험 |
| Outcomes | "수강 후 달라진 모습" | 변화 + 결과 |
| FAQ | "궁금한 점이 있으신가요?" | 친근함 |
| Final CTA | "지금 시작하지 않으면, 언제?" | 긴박감 |

### Level 3: Card Titles (구체적 가치)
- 3-5단어, 명사구/동사구
- 수치/통계 포함 시 신뢰도 상승

### Level 4: CTAs & Microcopy (행동 유도)

| 위치 | Primary CTA | Secondary CTA |
|------|-------------|---------------|
| Hero | "지금 무료로 시작하기" | "쇼케이스 둘러보기" |
| Curriculum | "커리큘럼 전체 보기" | "Chapter 1 무료 체험" |
| Showcases | "AI 에이전트 체험하기" | "프로젝트 상세 보기" |
| Final CTA | "첫 챕터 무료 시작" | "커리큘럼 전체 보기" |

## 3. 카피 톤 & 매너

### 언어 레지스터: Approachable Professional (친근한 전문가)
- 존댓말 사용: "~습니다", "~하세요"
- 긍정적 명령형: "만들어보세요", "체험하세요"
- 과도한 겸손 회피

### 기술 어휘: Explain-First, Then Jargon
- 첫 언급 시 설명 추가: "RAG(검색 증강 생성)"
- Apple.com 스타일 은유: "벡터DB" → "의미를 이해하는 검색 엔진"

### 감정 트리거 균형
- **FOMO (10%)**: Final CTA에만 — "지금 시작하지 않으면, 1년 후에는 더 어려워집니다"
- **Aspiration (60%)**: Hero, Outcomes — "API 호출 → 자율 에이전트 설계"
- **Capability (30%)**: Value Props, FAQ — "Python 기초만 있으면 충분합니다"

```
CTA Primary: "지금 무료로 시작하기"
CTA Secondary: "쇼케이스 체험하기"
Guarantee: ✓ Chapter 1 무료 체험  ✓ 무기한 접근  ✓ 커뮤니티 Q&A 지원
```

### 4.2 Why Section (문제 → 해결)

**Problem Cards (3개)**
1. 🤔 진입 장벽: "ChatGPT API 호출과 자율 에이전트 설계는 완전히 다른 영역"
2. 💼 실무 갭: "단순 API 호출만으로는 실무에서 통하지 않습니다"
3. ⏰ 시간 부족: "혼자서 공부하려면 6개월이 걸립니다"

**Solution Cards (3개)**
1. 📚 체계적 커리큘럼: "기초부터 배포까지, 8개 챕터"
2. 🛠️ 실전 프로젝트: "이론에 그치지 않는 6개 실전 프로젝트"
3. ⚡ 최신 기술: "LangChain, n8n, MCP 프로토콜"

### 4.3 Value Props (3 Cards)

**Card 1**: 🧠 "단 하나의 커리큘럼이면 충분합니다" — 8 챕터, 33시간
**Card 2**: 🛠️ "포트폴리오가 바로 만들어집니다" — 6 프로젝트
**Card 3**: ⚡ "2025년 현재 가장 주목받는 기술" — 50+ 실습

### 4.4 FAQ (8개 Q&A)

1. **프로그래밍 경험이 없어도?** → Python 기초만 있으면 충분
2. **수강 기간?** → 33시간, 주 5시간 기준 7주
3. **쇼케이스 체험 가능?** → 회원가입 없이 데모 제공
4. **n8n vs LangChain 순서?** → Ch1-3 기초 → Ch4 n8n
5. **MCP 프로토콜이란?** → AI가 외부 도구에 접근하는 표준
6. **질문은 어디서?** → 커뮤니티 Q&A, AI 봇, 월 2회 라이브
7. **수료증?** → 8챕터 완료 + 6프로젝트 제출 시 발급
8. **환불 정책?** → 7일 100% / 14일 50%

```
Body: 첫 챕터는 무료로 체험할 수 있습니다.

CTA Secondary: "커리큘럼 전체 보기"
```

## 5. 콘텐츠 밀도 (Apple.com 패턴)

| 섹션 | 밀도 | 적정 단어 수 |
|------|------|------------|
| Hero | 초희박 | 30-40단어 |
| Why | 중간 | 카드당 40-60단어 |
| Value Props | 중간 | 카드당 50-70단어 |
| Curriculum | 밀집 | 카드당 80-120단어 |
| Personas | 중간 | 카드당 60-80단어 |
| Showcases | 희박 | 카드당 30-50단어 |
| FAQ | 밀집 | Q&A당 80-150단어 |
| Final CTA | 초희박 | 20-30단어 |

**핵심 원칙**:
1. "One Big Thing per Section" — 각 섹션 하나의 메시지
2. "Text as Visual Element" — 짧은 임팩트 문구
3. "Progressive Disclosure" — 기본 2줄 → 클릭 시 확장
4. "Whitespace > Text" — 화면 60% 여백, 텍스트 max-w 600px
